# React Blog With Firebase
Simple React blog with Firebase.

## Installation

Change the info on 'firebase.config.jsx' inside 'src/config' and add your own credentials

In the main folder, run
```bash
  yarn dev
```
If you ran into any issue, run
```bash
  yarn
```
And then
```bash
  yarn dev
```
